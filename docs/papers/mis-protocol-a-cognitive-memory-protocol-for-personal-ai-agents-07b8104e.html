Abstract / summary

MiS Protocol is a prototype cognitive memory architecture for long-term personal AI agents. It addresses the signal lost before language forms, its reconstruction into traceable Events, and the long-term governance of agent memory through a closed rational loop (Hard Loop) and a separate reflective growth layer.
